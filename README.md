16、排序算法的比较
【问题描述】比较多种排序算法的性能。
【基本要求】分别生成元素升序、降序、随机序的3个数组，长度都为100万。分别实现简单选择排序、希尔排序、直接插入排序、冒泡排序、归并排序、快速排序、堆排序、基数排序等算法。用这些算法分别对上述3个数组进行排序，用程序测量排序所用的时间，写成详细的测试和分析报告。
要求如下：
（1）允许使用的编程语言有C、C++、Java，关键数据结构需自己实现，底层排序、查找等算法自己写，不能调库函数，不能使用高级类库；
（2）信息管理类程序需设计前后台部分，前台供用户使用，后台供管理员维护数据用。
（3）数据应进行保存，使用文件存储数据，不能使用数据库；
（4）鼓励采用多种方案按从低效到高效的顺序解决同一个问题；
（5）鼓励实现的功能超过题目的基本要求；鼓励采用新技术手段解决问题；
